# DAO in Q Blockchain

Starting project with typescript hardhat boilerplate

# Dependency
Node V20+

## Instruction
- Compile:
```
npm run compile
```
- Run test:
```
npm run test
```
- Deploy
```
npm run deploy:<network>
```
- Verify on etherscan
```
npx hardhat verify --network rinkeby <YOUR_CONTRACT_ADDRESS>
```
